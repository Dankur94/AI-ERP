# Modul 3 — Matching / Abgleich

**Was:** Zwei Datensaetze automatisch vergleichen. Abweichungen markieren.

**Beispiele:** Rechnung ↔ PO, Rechnung ↔ Bankbewegung, Bestellung ↔ Lagerbestand.

## Scope

- [x] Dokument A mit Dokument B verknuepfen
- [x] Automatischer Feldvergleich
- [x] Abweichungen rot markieren

## Status

Gebaut (2026-04-25). Backend + Frontend funktionsfaehig.

## Technische Details

- Backend: POST/GET/DELETE /api/matches
- Matching-Service: Feldvergleich (Header + Line Items)
- Positionen: Exaktes + partielles Matching (Beschreibung)
- Status: match, mismatch, missing_a, missing_b
- Frontend: /matches Route, Dropdown-Auswahl, Vergleichstabelle
- L3: Alle Matches tenant-isoliert

## Spaeter

- Fuzzy-Matching fuer Beschreibungen (Levenshtein/AI)
- Toleranzen fuer Betraege (z.B. +/- 1%)
- Automatisches Matching bei Upload (PO-Nummer erkennen)
- Batch-Matching mehrerer Rechnungen
