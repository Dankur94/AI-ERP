# Woche 1 — PDF + Claude + JSON

## Ziel

Ein Script das eine Rechnung als PDF nimmt und strukturierte Daten zurueckgibt.

## Konkret

```
Input:  rechnung-lieferant-xyz.pdf
Output: {
  "lieferant": "Shenzhen Electronics Co.",
  "rechnungsnr": "INV-2026-0042",
  "datum": "2026-04-15",
  "waehrung": "USD",
  "betrag_gesamt": 4850.00,
  "positionen": [
    { "beschreibung": "USB-C Adapter", "menge": 500, "einzelpreis": 3.20 },
    { "beschreibung": "HDMI Kabel 2m", "menge": 300, "einzelpreis": 5.50 }
  ]
}
```

## Was man braucht

1. Claude API Key (anthropic.com)
2. Ein PDF einer echten Rechnung
3. ~30 Zeilen Code

## Checkliste

- [ ] API Key besorgt
- [ ] Beispiel-Rechnung als PDF da
- [ ] Script geschrieben: PDF → base64 → Claude → JSON
- [ ] Output ist korrekt (manuell geprueft)
- [ ] Mit 3 verschiedenen Rechnungen getestet
