# Woche 2 — Upload-Seite

## Ziel

Eine simple Webseite: PDF hochladen, Ergebnis sehen.

## Konkret

```
Browser:
  [Datei waehlen] [Hochladen]

  → Ladebalken...

  → Tabelle:
    Lieferant: Shenzhen Electronics Co.
    Rechnungsnr: INV-2026-0042
    Betrag: 4,850.00 USD
    Positionen: 2 Stueck
```

## Was man braucht

1. Einfacher Webserver (Express.js oder FastAPI)
2. Upload-Formular (1 HTML-Seite)
3. API-Endpoint der Woche-1-Script aufruft
4. Ergebnis als Tabelle anzeigen

## Kein:
- Login
- Design
- Responsive Layout
- Error Handling fuer Edge Cases

- [ ] Webserver laeuft lokal
