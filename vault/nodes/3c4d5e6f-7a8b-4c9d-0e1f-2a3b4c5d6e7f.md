# Command Palette & Fuzzy-Launcher

## Beispiele
- **VS Code:** Ctrl+Shift+P
- **Linear, Notion:** Cmd+K
- **macOS:** Spotlight, Alfred, Raycast
- **Superhuman:** Cmd+K fuer E-Mail-Aktionen

## Prinzip

Vereint die Tastatur-Geschwindigkeit der T-Codes mit Entdeckbarkeit. Man muss das Kuerzel nicht kennen — eine unscharfe Suche ueber semantisch beschriftete Aktionen reicht.

## Warum das der groesste praktische Fortschritt ist

Das ist vermutlich der groesste praktische Fortschritt gegenueber CICS/SAP-T-Codes:
- **Gleiche Geschwindigkeit** wie T-Codes fuer Experten (wenige Tastaturanschlaege)
- **Keine Auswendiglernen-Steuer** — wer das Kuerzel vergisst, tippt ein Wort
- **Entdeckbarkeit** — neue Funktionen tauchen in der Suche auf
- **Progressive Disclosure** — Anfaenger und Experten nutzen denselben Mechanismus

## SAP-Vergleich

| Dimension | SAP T-Code | Command Palette |
|---|---|---|
| Geschwindigkeit (Experte) | Sehr hoch | Sehr hoch |
| Geschwindigkeit (Anfaenger) | Niedrig (nachschlagen) | Hoch (tippen + suchen) |
| Entdeckbarkeit | Keine | Hoch |
| Memorability | Schlecht | Nicht noetig |
| Kontextbewusstsein | Keins | Moeglich (filtern nach Kontext) |

## Design-Anforderung fuer AI ERP

- Globale Command Palette mit Ctrl+K / Ctrl+Shift+P
- Fuzzy-Matching ueber alle Aktionen, Module, Datensaetze
- Kontextfilter: Aktionen anzeigen, die im aktuellen Zustand sinnvoll sind
- Tastenkuerzel fuer Top-20-Aktionen parallel beibehalten
- Suchhistorie + Frecency-Ranking (haeufig + kuerzlich = oben)
