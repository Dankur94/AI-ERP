# Wettbewerbsanalyse & Positionierung

## Das Produkt in einem Satz

**Dokument rein → extrahieren → abgleichen → reporten → warnen — eine Pipeline, ein Tool, fuer kleine Firmen.**

## Was existiert vs. was fehlt

| Existiert | Anbieter | Was FEHLT |
|---|---|---|
| AI PDF-Extraktion | Tofu, Klippa, Reducto | **Extraktion + Matching in EINEM Tool** |
| Buchhaltungs-Software | Xero, QuickBooks | **Branchenuebergreifend konfigurierbar** |
| Enterprise-Matching | SAP Concur | **Matching fuer 8-Leute-Firmen** |
| Einzelne Alerts | In jedem Tool ein bisschen | **Alles verbunden in einer Pipeline** |

## Unsere Position

Jeder Wettbewerber loest genau EIN Modul. Der Kunde steckt heute 3-5 Tools zusammen:

```
Heute (Kunde):    Tofu + Excel + Xero + manueller Abgleich + Hoffnung
Wir:              Upload → Extraktion → Suche → Matching → Report → Alert → Dashboard
                  EIN Login. EIN Preis. EINE Pipeline.
```

## Wettbewerb in 3 Worten

| Wer | Was sie tun | Luecke | Unsere Antwort |
|---|---|---|---|
| **Tofu** | Rechnung → Daten | Hoert danach auf | Wir machen weiter: Matching, Report, Alert |
| **SAP Concur** | Rechnung = Bestellung? | Nur Konzerne | Wir machen das fuer 8-Leute-Firmen |
| **Xero** | Wieviel Geld hab ich? | Nur Buchhaltung | Wir sind branchenuebergreifend |
| **Yonyou** | Alles in einem | Komplex, teuer, schwer | Wir sind einfach, guenstig, sofort nutzbar |

## 6 Module — eine Pipeline

```
1. Erfassen → 2. Finden → 3. Abgleichen → 4. Report → 5. Warnen → 6. Dashboard
```

Jedes Modul baut auf dem vorherigen auf. Nach jedem Modul kann verkauft werden.

## Warum wir gewinnen

1. **Kein Wettbewerber hat die volle Pipeline fuer KMU** — die Luecke ist real
2. **Branchenuebergreifend** — selbes Tool fuer Trading, Fashion, F&B, Restaurant, Healthcare
3. **Sofort nutzbar** — kein 6-Monats-Setup wie bei SAP, kein IT-Team noetig
4. **AI-native** — nicht AI draufgeklebt, sondern AI als Kern (Extraktion, Matching, Reports)
