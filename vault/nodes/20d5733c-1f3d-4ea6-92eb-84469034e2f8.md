# Gundam Kids 3D — Zeichnen & Drucken

## Idee

Kinder zeichnen mit Stift (Tablet/iPad) eigene Gundam-Teile. Die App macht daraus 3D-Modelle fuer den 3D-Drucker.

## Zielgruppe

- Kinder 6-14 Jahre
- Eltern mit 3D-Drucker (Hobby/Maker-Szene)
- Gundam/Mecha-Fans

## Kernablauf

```
Zeichnen (Stift) → Glaetten → 3D-Extrusion → Vorschau → STL-Export → 3D-Druck
```

1. Kind zeichnet Umriss auf Canvas (2D)
2. App glaettet Linien automatisch (kindgerecht)
3. Umriss wird zu 3D-Koerper extrudiert
4. Kind passt an: Tiefe, Rundung, Skalierung per Schieberegler
5. 3D-Vorschau zum Drehen/Zoomen
6. Export als STL-Datei fuer Slicer/3D-Drucker

## Features

| Feature | Prioritaet | Beschreibung |
|---|---|---|
| Stift-Zeichenflaeche mit Druckerkennung | Hoch | Canvas + Pointer Events API |
| 2D → 3D Extrusion | Hoch | Umriss wird zu druckbarem Koerper |
| 3D-Vorschau | Hoch | Drehen, Zoomen, Farbe aendern |
| STL-Export | Hoch | Direkt druckbare Datei |
| Teile-Bibliothek | Mittel | Vorgefertigte Gundam-Basisteile (Arme, Beine, Kopf) |
| Schieberegler | Mittel | Tiefe, Breite, Rundung anpassen |
| Teile kombinieren | Mittel | Mehrere Zeichnungen zu einem Modell |
| AI-Verbesserung | Niedrig | AI glaettet/verbessert Kinderzeichnung zu sauberem 3D |

## Tech-Stack

| Komponente | Technologie |
|---|---|
| App-Typ | Web-App (laeuft auf allen Geraeten mit Browser) |
| Frontend | Vue.js oder React |
| Zeichenflaeche | HTML Canvas + Pointer Events API |
| 3D-Engine | Three.js |
| 3D-Export | three-stl-exporter |
| Optional spaeter | PWA fuer Offline-Nutzung, AI fuer Form-Optimierung |

## Warum Web-App statt Native

- Ein Codebase fuer iPad, Android-Tablet, Surface
- Kein App-Store noetig
- Stift/Pointer Events funktionieren browseruebergreifend
- Schneller gebaut und deployed
- Spaeter als PWA installierbar

## Herausforderungen

1. **2D → 3D Qualitaet** — Einfache Extrusion gut fuer flache Teile. Organische Formen brauchen erweiterte Logik oder AI.
2. **Druckbarkeit** — STL muss wasserdicht (manifold) sein. Validierung noetig.
3. **Kindgerechte UX** — Grosse Buttons, wenig Text, visuelle Hinweise, Undo/Redo leicht erreichbar.

## Machbarkeit

Komplett mit Claude Code baubar. MVP: Zeichnen + Extrudieren + Exportieren.
