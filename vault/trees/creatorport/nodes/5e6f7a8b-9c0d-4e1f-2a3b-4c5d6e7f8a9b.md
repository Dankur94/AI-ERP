# Drone-Verbindung

## Wie App und Drone zusammenhaengen

Die App hat zwei Modi:
1. **Mecha-Modus** — Freie Figuren (Spielzeug zum Mitnehmen)
2. **Shell-Modus** — Drone-Gehaeuse (passt auf CreatorPort Drone-Kit)

Beide nutzen dasselbe Snap-Together-System. Das Kind merkt den Unterschied kaum.

## Drone-Hardware

- **Basis:** Bestehende Kits modifiziert (nicht komplett eigenes Design)
- **Spaeter:** Eigenes CreatorPort Drone-Kit
- **Vertrieb:** Im Workshop genutzt UND separat weltweit verkauft

## Technische Anforderungen an die Shell

| Parameter | Wert | Grund |
|---|---|---|
| Max. Gewicht Shell | ~30-50g | Flugfaehigkeit erhalten |
| Material | PLA / PETG | Leicht, druckbar, guenstig |
| Befestigung | Snap-Clips | Keine Schrauben, kindersicher |
| Motor-Aussparungen | Fix vordefiniert | Muessen exakt zum Kit passen |
| Prop-Clearance | Mindestabstand | Propeller duerfen nicht anstossen |

## Sicherheit

- Shell darf Propeller nicht blockieren
- App validiert Design automatisch vor Export
- "Rote Zonen" in der App die nicht bebaut werden duerfen
- Gewichtslimit-Anzeige: "Dein Design wiegt X Gramm — OK / zu schwer"

## Geschaeftsmodell

```
Workshop (HK)           → Kurs-Gebuehr inkl. Kit + App-Zugang
Drone-Kit (weltweit)    → Hardware-Verkauf + App gratis dazu
App allein              → Gratis (Lead-Gen fuer Kit-Verkauf)
Premium-Teile in App    → In-App-Kauf (spaeter)
```
