# Workshop-Flow

## Tagesablauf (3-4 Stunden)

### Phase 1: "Design deinen Mecha-Piloten" (45 min)
- Kind oeffnet App auf Tablet
- Guided Tour: Waehle Kopf → Torso → Arme → Beine → Waffen
- Farben und Details anpassen
- Ein Design wird zum Drone-Gehaeuse erklaert
- Export → 3D-Drucker startet im Hintergrund

### Phase 2: "Baue deine Drone" (60 min)
- Kids bauen Drone aus CreatorPort Hardware-Kit
- Standard-Teile: Motoren, Propeller, Flight Controller, Batterie
- Anleitung Schritt fuer Schritt (printed oder in-App)
- Waehrend sie bauen, druckt der 3D-Drucker ihr Gehaeuse

### Phase 3: "Zusammenbau" (30 min)
- 3D-Druck ist fertig
- Kind setzt selbst-designtes Gehaeuse auf Drone
- Snap-Together: keine Werkzeuge noetig
- Optional: Mecha-Figur wird auf/an der Drone befestigt

### Phase 4: "Testflug" (45 min)
- Jedes Kind fliegt seine eigene Drone
- Mit EIGENEM Design → Emotional wertvoller als Fertigprodukt
- Foto/Video fuer Eltern (Social Media Moment)
- "Ich hab das designt UND gebaut"

## Hardware-Bedarf pro Workshop

| Item | Anzahl | Notiz |
|---|---|---|
| Tablets (Android/iPad) | 1 pro Kind | Alternativ: Kids bringen eigene mit |
| 3D-Drucker | 2-3 | Parallel drucken, Bambu Lab P1S empfohlen |
| Drone-Kits | 1 pro Kind | CreatorPort Hardware-Kit |
| Filament | PLA, verschiedene Farben | ~50g pro Kind |

## Zeitplan realistisch?

- 3D-Druck braucht 30-60 min fuer kleine Teile
- Waehrend Phase 2 (Drone bauen) druckt der Drucker
- Bei 10-15 Kids und 3 Druckern: machbar wenn Teile klein genug sind
- Backup: Vorgedruckte Standard-Teile falls Drucker zu langsam
