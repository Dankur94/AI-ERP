# Setup & Deploy

## Lokales Setup

```bash
# 1. WhatsApp Bridge starten
cd C:\Projects\SOPTree\whatsapp-bridge
npm install
node bridge.js
# QR scannen: http://localhost:3006/qr

# 2. Python Backend starten
cd C:\Projects\SOPTree
pip install -r requirements.txt
cp config.example.yaml config.yaml
# Deadlines in config.yaml anpassen
uvicorn server:app --port 8006 --reload
```

## Deploy auf VM

```bash
cd C:\Projects\SOPTree && bash deploy.sh
```

Das Script macht: git push → VM pullt → Service restart → Health-Check.

## Erster Deploy (VM einrichten)

1. GitHub-Repo erstellen: `gh repo create SOPTree --private --source=. --remote=origin`
2. Pushen: `git push -u origin main`
3. `deploy.sh` ausfuehren (klont automatisch auf VM)
4. Auf VM config.yaml anlegen: `ssh VM 'cd /home/ubuntu/soptree && cp config.example.yaml config.yaml && nano config.yaml'`
5. Systemd Service einrichten (wie bei anderen Bots)

## Desktop-Shortcut

`C:\Users\User\Desktop\SOPTree.bat` — Oeffnet Claude Code im SOPTree-Projekt.

## Deadlines aendern

In `config.yaml` unter `modules.deadline.deadlines`:
```yaml
deadlines:
  - name: "HKSTP Ideation"
    deadline: "2026-05-18T12:00"
  - name: "Neuer Fond"
    deadline: "2026-12-31"
  - name: "Laufendes Programm"
    deadline: null
```

Nach Aenderung: Bot neu starten.
