# SOP: WhatsApp Bot (SOPTree)

## Was ist das

WhatsApp-Bot fuer die Gruppe "SOP Tree". Gleiche Architektur wie FamilyHub, 37TabsInHerBrain, HeartSynch, HoK-RoadtoGlory.

## Was kann der Bot

| Modul | Befehl | Funktion |
|---|---|---|
| **deadline** | `deadline` / `deadlines` | Zeigt alle Funding-Deadlines mit verbleibenden Tagen |
| **help** | `help` | Zeigt verfuegbare Befehle |

### Automatisch (Scheduler)
- **Taeglich 10:00 Uhr**: Sendet Funding-Deadline-Uebersicht in die WhatsApp-Gruppe

## Architektur

```
WhatsApp Gruppe "SOP Tree"
        |
  bridge.js (Port 3006, Baileys)
        |  HTTP
  server.py (Port 8006, FastAPI + APScheduler)
        |
  Module (deadline, help, ...)
        |
  SQLite (soptree.db)
```

## Starten

```bash
# Terminal 1: WhatsApp Bridge
cd C:\Projects\SOPTree\whatsapp-bridge && npm install && node bridge.js
# Dann QR scannen: http://localhost:3006/qr

# Terminal 2: Python Backend
cd C:\Projects\SOPTree
pip install -r requirements.txt
cp config.example.yaml config.yaml
uvicorn server:app --port 8006 --reload
```

## Deploy (VM)

```bash
cd C:\Projects\SOPTree && bash deploy.sh
```

## Neue Deadlines hinzufuegen

In `config.yaml` unter `modules.deadline.deadlines`:
```yaml
- name: "Neuer Fond XY"
  deadline: "2026-12-31"
```

Bot muss danach neu gestartet werden.
