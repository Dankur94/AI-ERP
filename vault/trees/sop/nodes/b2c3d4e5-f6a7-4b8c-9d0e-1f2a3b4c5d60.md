# Architektur & Ports

## SOPTree Bot

- **Projekt:** `C:\Projects\SOPTree\`
- **Bridge:** Port 3006 (Node.js, Baileys v6)
- **Backend:** Port 8006 (Python, FastAPI)
- **Datenbank:** SQLite (`soptree.db`)
- **WhatsApp-Gruppe:** SOP Tree
- **VM:** `ubuntu@13.60.99.119` → `/home/ubuntu/soptree/`
- **Service:** `soptree-bot.service`

## Architektur-Pattern (identisch bei allen Bots)

```
WhatsApp Gruppe
    |
bridge.js (Baileys, eigener Port)
    | HTTP POST /message
server.py (FastAPI + APScheduler)
    |
registry.py → route_message() → modules
    |
database.py (SQLite)
```
