## Schmerz

*"My food is great. But nobody knows we exist. I can't afford influencers, and I can't make good content myself. Tuesday to Thursday I'm bleeding money."*

## Das Platform-Modell — Win-Win-Win

Jennys Problem ist nicht IT im klassischen Sinn. Ihr Problem ist **Sichtbarkeit**. Die Loesung ist eine **Platform die drei Seiten verbindet:**

### Wie es funktioniert:

```
┌─────────────┐     ┌──────────────────┐     ┌─────────────────┐
│  RESTAURANT  │     │    PLATFORM      │     │   INFLUENCER    │
│  Jenny       │────→│  (dein Produkt)  │←────│   Content       │
│              │     │                  │     │   Creator       │
│ Bietet:      │     │ Bietet:          │     │ Bietet:         │
│ - Essen      │     │ - Camera Setup   │     │ - Reichweite    │
│ - Location   │     │ - Video-Dreh     │     │ - Followers     │
│ - Story      │     │ - Matching       │     │ - Content-Skill │
│              │     │ - Analytics      │     │                 │
│ Bekommt:     │     │ Bekommt:         │     │ Bekommt:        │
│ - Content    │     │ - Abo-Gebuehr    │     │ - Free Dining   │
│ - Reichweite │     │ - Daten          │     │ - Pro Content   │
│ - Gaeste     │     │ - Netzwerkeffekt │     │ - Neues Publikum│
└─────────────┘     └──────────────────┘     └─────────────────┘
```

### Win 1 — Restaurant (Jenny):
- Professioneller Video/Foto-Content ohne HK$15.000 pro Influencer
- Camera-Setup steht dauerhaft im Restaurant (monatliches Abo)
- Passende Micro-Influencer werden automatisch eingeladen
- Daten: Welcher Content bringt tatsaechlich Gaeste (nicht nur Likes)

### Win 2 — Influencer:
- Kostenloses Essen + Erlebnis
- Professionelles Setup vor Ort (besserer Content als mit Handy)
- Neue Restaurants entdecken bevor sie mainstream sind
- Portfolio-Aufbau mit hochwertigem Material

### Win 3 — Platform (du):
- Monatliches Abo von Restaurants fuer Camera-Setup + Matching
- Daten ueber Gaeste-Verhalten, Content-Performance, Conversion
- Netzwerkeffekt: Mehr Restaurants → mehr Influencer → mehr Restaurants
- Community bildet sich organisch um die Platform

## Daten die dabei entstehen (legal, ohne zu fragen)

- Welcher Content-Typ bringt Reservierungen (Video vs. Foto vs. Reel)
- Welcher Influencer bringt tatsaechlich Gaeste (Conversion, nicht nur Views)
- Optimale Posting-Zeiten fuer HK Food-Content
- Unter-der-Woche-Muster: Was fuellt den Dienstag?
- Saisonale Trends: Welche Gerichte performen wann
- Preis-Sensitivitaet: Welche Angebote ziehen, welche nicht
