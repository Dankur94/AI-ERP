## AI-Strategie

```
Stufe 1 (jetzt):   Claude API         ~$0.01-0.02/Rechnung
Stufe 2 (12M):     Immer noch API     Marge steigt mit Volumen
Stufe 3 (24M):     Fine-tuning        Claude mit unseren Daten anpassen → billiger
Stufe 4 (48M):     Eigenes Modell     Nur wenn Millionen Dokumente/Monat
```

Wettbewerber (Tofu, SAP, Xero, Yonyou) trainieren eigene Modelle — weil sie gross sind.
Fuer uns in Stufe 1: API ist 100x schlauer. Kein ML-Team, sofort einsatzbereit, wechselbar.

## Trainingsdaten-Strategie (Moat)

Jede Kundenkorrektur = kostenloses Trainingsmaterial.
- Von Tag 1 speichern wir: Original-PDF + extrahierte Daten + Korrekturen
- Spaeter nutzbar fuer Fine-tuning oder eigenes Modell
- Rechtlich: Ein Satz in ToS reicht ("anonymized data to improve accuracy")
- Burggraben: Nach 2 Jahren haben wir 100.000+ PDFs. Neuer Wettbewerber braucht Jahre.
