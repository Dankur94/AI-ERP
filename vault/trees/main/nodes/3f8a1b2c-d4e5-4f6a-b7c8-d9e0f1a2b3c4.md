# Schutzschichten — Qualitaet ohne Gedaechtnis

Claude startet jeden Tag ohne Erinnerung. Ohne Schutz baut Claude jeden Tag ein anderes Produkt.

## 6 Gefahren → 4 Schichten

| Gefahr | Schicht die schuetzt |
|---|---|
| Doppelt bauen | Schicht 1 (CHANGELOG) + Schicht 2 (Handoff) |
| Widersprueche einbauen | Schicht 3 (Tests) |
| Architektur-Drift | Schicht 1 (Strukturregeln) |
| Lessons vergessen | Schicht 1 (L1-L4 Pflicht, L6/L9/L11/L15 Verbot) |
| Konventionen brechen | Schicht 1 (Regeln) + Schicht 3 (Linter) |
| Persona-Bezug verlieren | Schicht 1 (Personas laden) |

## Morgen-Routine

```
1. Claude startet → CLAUDE.md geladen (Schicht 1)
2. Claude liest Session-Handoff (Schicht 2)
3. Du sagst was du willst
4. Claude arbeitet
5. Tests + Linter laufen (Schicht 3)
6. Am Ende: neuer Handoff fuer morgen
```
