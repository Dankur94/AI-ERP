# Modul 4 — Automatische Reports

**Was:** Per Knopfdruck fertigen Bericht generieren. Formatiert, sortiert, versandfertig.

**Beispiele:** Zoll-Doku, Monatsabschluss fuer Steuerberater, Versicherungsbericht.

## Scope

- [x] Report-Summary (Anzahl, Summen, nach Status/Lieferant/Waehrung)
- [x] Daten aus Modul 1-3 zusammenfuehren
- [x] Export als CSV
- [ ] Report-Templates definierbar
- [ ] Export als PDF/Excel

## Status

Grundfunktion gebaut (2026-04-25). Summary + CSV-Export funktionsfaehig.

## Technische Details

- Backend: GET /api/reports/summary, GET /api/reports/export/csv
- KPI-Karten: Anzahl, Gesamtsumme, Lieferanten, Waehrungen
- Aufschluesselung: Status, Waehrung, Lieferant (Top 20)
- Datumsfilter fuer Summary und Export
- L3: Tenant-isoliert
- L4: CSV fuer Stufe 1

## Spaeter

- PDF-Export (reportlab oder weasyprint)
- Excel-Export (openpyxl)
- Report-Templates (Zoll, Steuerberater, etc.)
- Scheduled Reports (woechentlich/monatlich per E-Mail)
