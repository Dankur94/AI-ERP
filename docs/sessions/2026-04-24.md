# Session 2026-04-24

## Was gemacht

### Modul 1 — Dokumenten-Erfassung gebaut (ohne Claude API)

**Backend (src/api/):**
- FastAPI App mit Upload-Endpunkt, CRUD fuer Rechnungen
- SQLite Datenbank: invoices (inkl. 6 Confidence-Spalten) + line_items
- Mock-Extraktion: Shanghai Electronics Co. Dummy-Daten mit Confidence-Scores
- Alle Endpoints getestet: Upload, List, Get, Update, PDF-Download

**Frontend (src/web/):**
- Vue.js 3 + Vite + Vue Router
- Drag & Drop Upload-Zone (Multi-PDF)
- Split-View: PDF links (iframe), Formular rechts
- Confidence-Badges: Sicher (gruen), Pruefen (gelb), Unsicher (rot)
- Editierbare Felder + Positionen-Tabelle
- Speichern/Bestaetigen Buttons

**Infrastruktur:**
- Python venv mit Windows Python 3.12 (nicht Inkscape-Python!)
- Start-Scripts: start-backend.sh, start-frontend.sh
- Vite Proxy: /api -> localhost:8000
- Alles getestet und laeuft

### Claude API Kosten besprochen
- ~$0.01-0.02 pro Rechnung (Sonnet)
- ~$25/Monat bei 50 Rechnungen/Tag
- Kein Lock-in: Anbieter wechseln = 1 Datei aendern
